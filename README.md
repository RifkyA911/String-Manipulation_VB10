<h1 align="center">🔤 String Manipulation — VB10</h1>
<p align="center"><em>A simple string manipulation application built with Visual Basic 10</em></p>

---

## 🇬🇧 English

**String Manipulation — VB10** is a Windows desktop application that demonstrates various text processing operations.

### ✨ Features
- 🔠 Case conversion (uppercase, lowercase, toggle)
- ✂️ String trimming and concatenation
- 🔍 Substring search
- ✅ Input text validation
- 🖥️ Simple GUI interface

### 🚀 Getting Started
1. Open the project in **Visual Studio** (VB10)
2. Press Ctrl+Shift+B to build
3. Press F5 to run

### 🛠️ Tech Stack
- **Language:** Visual Basic 10
- **Framework:** .NET Framework
- **IDE:** Visual Studio 2010+
