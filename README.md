## English

A Windows desktop application that demonstrates various text processing operations built with **Visual Basic 10**.

### Features
- Case conversion (uppercase, lowercase, toggle)
- String trimming and concatenation
- Substring search
- Input text validation
- Simple GUI interface

### Getting Started
1. Open the project in **Visual Studio** (VB10)
2. Press Ctrl+Shift+B to build
3. Press F5 to run

### Tech Stack
- Language: Visual Basic 10
- Framework: .NET Framework
- IDE: Visual Studio 2010+
